# Call System Manual Checklist

1. Host toggles online.
2. Host toggles offline.
3. User sees host appear realtime.
4. User sees host disappear or become unavailable realtime.
5. User requests audio call.
6. User requests video call.
7. Receiver gets incoming call popup.
8. Receiver accepts.
9. Both join same LiveKit room.
10. Caller ends call.
11. Coins deducted correctly.
12. Earning ledger created.
13. Host earning calculated.
14. Agency earning calculated.
15. Platform earning calculated.
16. Receiver rejects call.
17. Missed call after timeout.
18. Caller with low coins cannot call.
19. Busy host cannot receive second call.
20. Admin sees call in dashboard.
21. Agency sees only own host calls.
22. CSV export works.
23. App disconnect during call handled.
24. Duplicate tap does not create duplicate calls.
